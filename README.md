# RGB Color Recognition with OpenCV

## Overview  
Simple Python program to detect and display the RGB color of any pixel clicked on an image.

## Features  
- Load and resize an image.  
- Click on the image to get pixel RGB color.  
- Show color rectangle with RGB values.  
- Exit with ESC key.

## Requirements  
- Python 3.x  
- OpenCV (`opencv-python`)

## Usage
- Put your image as image.jpg in the script folder.
- Run the script: python main.py
- Click on any pixel to see its RGB color.
- Press ESC to exit.

## How It Works
- Reads and resizes image.
- On mouse click, reads pixel color.
- Displays color and RGB on screen.
